# 🌍 Global Economic Dashboard (Version 4.7)

> An automated macroeconomic intelligence suite tracking GDP, Inflation, and Unemployment across major economies using the World Bank API.

## 🚀 Live Project
**[View the Interactive Dashboard](https://vishnurohith225-beep.github.io/global-economic-dashboard/global_economic_dashboard.html)**

## 🛠️ Key Features
- **Automated Data Pipeline**: Real-time data ingestion via `wbdata`.
- **Predictive Analytics**: 5-year projections (2024–2028) using Linear Regression.
- **Fiscal Health Scoring**: Proprietary Risk Matrix weighing Debt-to-GDP against interest burdens.
- **Interactive Visualizations**: Multi-panel Plotly dashboards and Trajectory plots.

## 🧰 Tech Stack
- **Language**: Python 3.10+
- **Libraries**: Pandas, Scikit-Learn, Plotly, Statsmodels, Seaborn, Matplotlib
- **Deployment**: GitHub Pages
